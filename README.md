## How to start

Clone the project:

```
git clone https://github.com/Tryoe/frontAibalst.git
```

Install dependencies, currently using pnpm, but other package managers should work too.

```
pnpm i
```

Start the project on development mode

```
pnpm dev
```

App should be running now.
